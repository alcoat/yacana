Roadmap

The Yacana adventure is merely just starting.

We will be working on these new features in the next months:

  • Compatibility with MCP
  • Allowing both Agents types to use enhanced tool calling and the OpenAI tool calling format.
  • Adding memory to Agents
  • Adding a RAG tool available for all agents
  • More to come

If there's a functionality that you need, please open an issue and we will gladly look into it!
